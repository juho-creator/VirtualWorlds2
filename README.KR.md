## 프로젝트의 한계

현재, 다른 동물이 이미 존재하는 위치에 동물이 생성될 때 충돌 방지가 발생하지 않을 수 있습니다.
</br></br>


## 향후 개선사항
다른 동물이 이미 존재하는 위치에 동물이 생성되지 않도록 방지
